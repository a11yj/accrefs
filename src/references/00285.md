NVDA 日本語チームのにしもつ氏の WAI-ARIA の紹介記事。WAI-ARIA は HTML 要素に対して支援技術に伝える情報を追加しセマンティクスを強化するもので、特に「ライブリージョン」ロールは、対象要素の内容が更新されたことをスクリーンリーダーに能動的に読み上げるために役立つという。他、アクセシビリティ・サポーテッド情報と UI オートメーションについても触れている。
