Web アクセシビリティの書籍として定番の地位を獲得しつつある「デザイニング Web アクセシビリティ」。この電子版が昨年末に出版されました。アクセシビリティを語る書籍として、電子版もアクセシブルにしなければならないというミッションの課題と解決策について、Web 制作者でもある著者が語っている。
